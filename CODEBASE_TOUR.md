# Codebase tour

T2A is a control plane for AI-driven software development: it orchestrates engineering tasks, dependencies, execution cycles, and agent workers so humans can define goals and review outcomes while the system coordinates the rest. The backend is written in **Go 1.25+** (module `github.com/AlexsanderHamir/T2A`), persists to **PostgreSQL** via GORM, and exposes a REST API plus SSE at **`cmd/taskapi`** (`go run ./cmd/taskapi`). A companion **TypeScript/React** SPA lives under `web/` (Vite, Node ≥20) and talks to that API during local dev via the scripts in `scripts/dev.ps1` / `scripts/dev.sh`.
